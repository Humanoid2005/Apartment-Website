## Run Locally

Clone the project

```bash
  git clone https://github.com/Humanoid2005/Basic-Blog-Website.git
```

Go to the project directory

```bash
  cd project-folder
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

To open the website, type the link given below

```bash
  http://localhost:5173/
```
